TechCities Outlook


TechCities Outlook is a data visualization project giving insight of the tech industry in some of the top and upcoming cities in North America.
Each city node is represented by a sports team from that city.
All city nodes are interactive and contain more info pertaining to that particular city.

All information used for this project is referenced from CBRE Tech Report (July, 2019). Report can be found here. Thanks for visiting!
